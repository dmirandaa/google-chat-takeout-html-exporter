✨ Key Features

  • ✅ Folder-per-conversation structure - Each conversation isolated in its own directory
  • ✅ Source-referenced attachments - No file copying; references point to original Google Chat/Groups data
  • ✅ Dark mode - Theme persists across page navigation via URL parameters
  • ✅ Responsive design - Works on desktop, tablet, and mobile
  • ✅ Timezone support - Timestamps formatted per your configured timezone
  • ✅ i18n support - Locale-based UI translations (en, pt-BR, extensible)
  • ✅ Material Design 3 - Modern, professional UI with smooth transitions
  • ✅ Gravatar avatars - Real user photos with fallback to colored initials
  • ✅ URL previews - Rich metadata cards for shared links
